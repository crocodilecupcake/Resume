# AI RESUME SYSTEM

## 🚀 Overview

This project focuses on building a **Business Intelligence & Predictive Data Analytics (BIra PDA) system** designed to analyze supply chain and operational data. It provides insights using KPIs, AI-based alerts, and data visualization techniques.

---

## 🎯 Objectives

* Analyze business data efficiently
* Generate meaningful **KPIs (Key Performance Indicators)**
* Detect anomalies like low stock and delayed shipments
* Provide actionable insights using AI

---

## 🧩 Features

### 🔢 KPI Calculation

* Calculates **total orders, revenue, and stock levels**
* Identifies **delayed shipments**
* Evaluates **supplier performance and ratings**
* Computes **demand-supply gaps**

### 🤖 AI-Based Alert Generation

* Detects **low stock levels**
* Identifies **shipment delays and supplier issues**
* Generates **recommendations for decision-making**

### 📊 Data Visualization

* Uses charts like:

  * Bar graphs
  * Line charts
  * Pie charts
* Displays trends and comparisons clearly

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/your-repo.git
```

2. Navigate to the project folder:

```bash
cd your-repo
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
streamlit run app.py
```
